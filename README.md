# 📊 Food Delivery Performance Dashboard (Power BI)

📌 Project Overview
This project presents an **interactive Power BI dashboard** that analyzes food delivery business performance across multiple cities, cuisines, and restaurants.
The dashboard helps stakeholders monitor **order trends, revenue performance, cuisine demand, and delivery efficiency**, enabling better data-driven decision-making.

---

 📈 Key Metrics

* **Total Orders:** 10,000+
* **Total Revenue:** ₹6.57 Million
* **Average Customer Rating:** 3.9 ⭐
* **Average Delivery Time:** 41 minutes

---

# 📊 Dashboard Features

### 1️⃣ Orders Trend

Analyzes monthly order patterns to identify demand fluctuations throughout the year.

### 2️⃣ Cuisine Popularity

Compares order volumes across different cuisines to identify the most popular food categories.

### 3️⃣ Top Cities by Orders

Highlights cities generating the highest number of orders.

### 4️⃣ Top Restaurants by Revenue

Shows restaurants contributing the most to overall revenue.

### 5️⃣ Veg vs Non-Veg Distribution

Provides insights into customer dietary preferences.

### 6️⃣ Delivery Efficiency Analysis

A scatter plot analyzing the relationship between **delivery time and customer ratings** across cities.

### 7️⃣ City vs Cuisine Matrix

A heatmap-style matrix comparing cuisine popularity across different cities.

### 8️⃣ Interactive Filters

Users can filter the dashboard using:

* City
* Cuisine
* Order Status

### 9️⃣ Advanced Tooltips
Custom tooltip pages provide additional KPI insights when hovering over visuals.

# 💡 Key Business Insights

* **Ahmedabad and Bangalore generate the highest order volumes.**
* **Mughlai and Andhra cuisines are among the most popular.**
* Cities with **lower delivery times tend to have slightly higher customer ratings.**
